Views

1) Create a view that shows all of the customers who have the highest ratings.

Ans. 	CREATE VIEW high_rating AS
	SELECT *
	FROM customers
	WHERE rating=
        (SELECT max(rating)
    	FROM customers);

2) Create a view that shows the number of salespeople in each city.

Ans. 	CREATE VIEW sp_city AS
	SELECT city,
       	count(city)
	FROM salespeople
	GROUP BY city;


3) Create a view that shows the average and total orders for each salesperson after
his or her name. Assume all names are unique.

Ans. 	CREATE VIEW avg_total_sp AS
	SELECT sname,
       	avg(onum) AVERAGE,
       	count(onum)
	FROM salespeople
	JOIN orders using(snum)
	GROUP BY sname;

4) Create a view that shows each salesperson with multiple customers.

Ans. 	CREATE VIEW multi_cus AS
	SELECT sname,cname
	FROM salespeople
	JOIN customers USING (snum);
